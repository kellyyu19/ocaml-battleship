val init_state : state

init_state is a new state, i.e. a new battleship game.

exception OutOfBounds

OutOfBounds is raised when the given coordinates are outside of the grid or do not correspond to ship size.

exception NotRight

NotRight is raised when the given coordinates could not possibly make a ship because they are neither part of the same column nor the same row.

place ship coordOne coordTwo state is the new state of the game when ship is placed in the grid given by state. Raises: NotRight is coordinates are incompatible. OutOfBounds if coordinates are outside of the grid.

val new_ship_list : Battleship.ship ->
       Battleship.ship list -> Battleship.ship list -> Battleship.ship list

new_ship_list ship ship_list outlist is the list of ships resulting from finding ship in ship_list and incrementing its hit count.

val sink_ship : Battleship.ship ->
       Battleship.grid ->
       ((char * int) * Battleship.status) list ->
       ((char * int) * Battleship.status) list

sink_ship ship currentGrid outlist is the grid that has located ship and changed every point representing ship to the Sunk status.

val update_grid_occupied : Battleship.ship ->
       char * int ->
       state ->
       Battleship.grid -> Battleship.point list -> Battleship.point list

upgrade_grid_occupied ship coord state currentGrid outlist is currentGrid with points containing ship changed from Occupied status to Hit status.

val update_grid_empty : char * int ->
       Battleship.grid -> Battleship.point list -> Battleship.point list

upgrade_grid_empty coord currentGrid outlist is currentGrid with the point at the given coordinate changed from Empty status to Miss status.

val is_sunk : Battleship.ship -> bool

is_sunk ship is whether or not ship is of Sunk status.

val curr_sunk_list : Battleship.ship list -> Battleship.ship list -> Battleship.ship list

curr_sunk_list currShipList outlist is the list of ships that have sunk in the current game.

val fire : Battleship.coordinate -> state -> state

fire coord currentState is the new game state resulting from firing at the given coordinate.

val placing : state -> bool

placing is whether or not the current game is still in placing mode. i.e. have all the ships been placed?

val string_of_ships : Battleship.ship -> string

string_of_ships is a string of the names of each type of ship.

val queue_helper : state ->
       Battleship.ship list -> Battleship.ship list -> Battleship.ship list

queue_helper currentState initships outlist is the list of ships that still need to be placed.

val queue : state -> string

queue currentState is a well formatted, pretty string of the list of ships that still need to be placed.

val getAmountSunk : 'a list -> int -> int

getAmountSunk is the number of ships that have sunk in the current game.

val winOrNot : 'a list -> bool

winOrNot is whether or not all ships have sunk in this game.
