Malformed [Battleship]

Raised when there is an invalid command.

N
NotRight [State]

NotRight is raised when the given coordinates could not possibly make a ship because they are neither part of the same column nor the same row.

O
OutOfBounds [State]

OutOfBounds is raised when the given coordinates are outside of the grid or do not correspond to ship size.

S
ShipHere [Battleship]

Raised when a ship is already placed at given coordinates.
