type name =
| Carrier
| Battleship
| Cruiser
| Submarine
| Destroyer 
type ship = {
   name : name;
   size : int;
   hits : int;
}

The type ship that contains the name, size, and hits.

type coordinate = char * int

The type coordinate that is a tuple of a char and an int.

type status =
| Occupied of ship
| Hit of ship
| Sunk of ship
| Empty
| Miss

The type status that is one-of these.

type point = coordinate * status

The type point that is a tuple of a coordinate and a status.

type grid = point list

The type grid that is a point list

exception ShipHere

Raised when a ship is already placed at given coordinates.

exception Malformed

Raised when there is an invalid command.

val rows : char list
val columns : int list
val pair : char ->
       int list ->
       ((char * int) * status) list ->
       ((char * int) * status) list

pair r c outlist is a list of points that is formed with a tuple of a tuple of r and each element of cols and the status Empty. Requires: cols is not empty.

val init_grid : char list -> int list -> grid -> grid

init_grid r c outlist is the initial grid that is formed, with all points having the status Empty. Requires: r is not empty. c is not empty.

val make_new_int_list : char -> int list -> int -> int -> (char * int) list -> (char * int) list

make_new_int_list r c c1 c2 outlist is a list of coordinates, where each coordinate is a tuple of r and an integer between c1 and c2, inclusive. Requires: c is not empty.

val make_new_char_list : char list -> int -> char -> char -> (char * int) list -> (char * int) list

make_new_char_list r c c1 c2 outlist is a list of coordinates, where each coordinate is a tuple of a char between r1 and r2,inclusive, and an integer c. Requires: r is not empty.

val make_grid : ship ->
       (char * int) list ->
       ((char * int) * status) list -> grid -> grid

make_grid ship ship_coords grid outlist is a grid that is formed depending on the status of each point. Requires: ship_coords is not empty. grid is not empty. Raises: ShipHere if there is already occupied at a point where a new ship is trying to be placed Malformed when there is an invalid command.
